Pokedex project details

Objective

To buile a small web application with HTML, CSS and JavaScript that loads data from an external API and enavbles the viewing of datapoints in detail

Steps to build the project

Step One

  1. Creating a GitHub repository with the name Pokedex
  2. Creating the files index.html README.md styles.css and js file
  3. connect the files with each others
  4. commit to github

Step Two

  1. Create a new variable called pokemonList and assign to it a blank array. This array contains Pokémon data to display in the application
  2. add several objects to the array. Each object represents one Pokémon and needs to have the same keys to avoid errors
  3. commit to github

Step Three

  1. wrap your pokemonList array in an IIFE to avoid accidentally accessing the global state
  2. create a new pokemonRepository variable to hold what IIFE will return, then assign the IIFE to that variable.
  3. The IIFE should return an object with the following public functions assigned as keys: getAll: return all items (pokemonRepository.getAll(); should return the pokemonList array) add: add a single item to the pokemonList array (calling pokemonRepository.add(item); should add the Pokémon referred to with item to the pokemonList array)
  4. use one of the two functions returned by the IIFE in order to retrieve the pokemonList array
  5. commit to github

Step four

  1. create an addListItem() in IIFE. Using the function inside forEach() loop to create a button for each Pokémon in the array
  2. add an event listener to each newly created button for each Pokémon in the list
  3. commit to github

Step five

  1. Add polyfills for promises and fetch to the project
  2. Add functions inside data repository to load data from an external source
  3. commit to github

Step six

  1. Add Bootstrap to the project
  2. create a modal in js file
  3. commit to github

Step seven

  1. Edit the showDetails() method of the application to show a modal with the Pokémons name, its height, and an image of the Pokémon
  2. Use the proper Bootstrap utility classes in the application
  3. Replace the modal in “index.html” with the Bootstrap modal.
  4. commit to github

Final step

  1. Add an “.eslintrc” file to the root of the project
  2. Fix linting issues
  3. commit to github
